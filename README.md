# Saucedemo-UI-Automation-Cypress
This project validates the complete user journey, including secure authentication, product selection, shopping cart management, and the full checkout process.

Test Scenarios Covered;
 Open website and verify landing page.
 Search for [Product Name] and validate results.
 Add item to cart and verify cart count.
 Proceed to checkout and confirm the final step.

Setup & Installation
 Clone the repo.
 Run [Installation Command] to install dependencies.

How to Run
 Execute [Test Command] to run all tests.

Reports can be viewed in the /reports folder.
